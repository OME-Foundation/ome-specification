# Research record templates

Status: required fields for Phase 1 research records.

Templates are prompts for critical reasoning, not forms to complete
mechanically. Remove instructional comments when creating a record, but do not
remove a required field merely because the answer is unknown; write `Unknown`
and create or link the work needed to resolve it.

- [Claim template](CLAIM.md)
- [Evidence record template](EVIDENCE.md)
- [Research item template](RESEARCH_ITEM.md)
