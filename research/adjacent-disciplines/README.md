# Adjacent-disciplines review

Status: research workstream; not accepted OME canon.

## Purpose

This review tests whether the founding OME vocabulary describes genuinely
unaddressed problems, useful cross-disciplinary synthesis, or renamed versions
of established ideas. It is deliberately critical. A term is not protected
because it appeared in the founding conversation.

> No concept enters the OME canon until its relationship to adjacent
> disciplines has been documented.

Claims use the repository-wide
[claim-classification standard](../CLAIM_CLASSIFICATION.md).

## Method

The review prioritises original papers, standards bodies, and framework owners.
Each discipline note separates **Source findings** from **OME interpretation**.
The review is selective rather than exhaustive: it seeks the concepts that
most constrain OME's claims and design.

## Classification scheme

| Class | Meaning |
| --- | --- |
| Established | Recognised in prior literature or standards with substantially the same meaning. |
| Synthesis | A useful combination or application of established concepts. |
| Hypothesis | A falsifiable claim that currently lacks sufficient validation. |
| Principle | A normative design choice, not an empirical discovery. |
| Novel proposal | A proposed construct with no close equivalent found and a clear added purpose. |
| Open question | Materially unresolved; OME should not state it as fact. |

## Citation policy

Links identify the referenced version or publisher record. Findings attributed
to sources should not be read as endorsement of OME's interpretation. Where a
standard is paywalled, this review relies only on its public abstract and does
not infer unpublished normative detail.
